# SaaS Onboarding Flow Demo

This project demonstrates a simple SaaS onboarding process built with HTML, CSS, and JavaScript.

## Features

- User Signup Form
- API Request Handling
- Dummy Payment Page
- Subscription Plan Selection
- Dashboard Page

## Project Flow

Signup  
↓  
API Request  
↓  
Dummy Payment  
↓  
Subscription Plan  
↓  
Dashboard

## Technologies Used

- HTML
- CSS
- JavaScript


## Purpose

This project was built to practice API integration and frontend onboarding flow commonly used in SaaS applications.
